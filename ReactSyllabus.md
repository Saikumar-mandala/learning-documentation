# ⚛️ Comprehensive React Topics Syllabus

## 1. Core Concepts
- [x] **JSX**: Syntax, Expressions, Fragments
- [x] **Components**: Functional vs Class (Legacy)
- [x] **Props**: Passing data, Prop Types, Default Props
- [x] **State**: `useState`, Immutability
- [x] **Events**: Handling events, Synthetic Events
- [x] **Conditional Rendering**: Ternary, `&&`, `if-else`
- [x] **Lists & Keys**: Rendering arrays, unique keys
- [x] **Forms**: Controlled vs Uncontrolled components

## 2. React Hooks (The Modern Way)
### Basic Hooks
- [x] `useState`: State management
- [x] `useEffect`: Side effects
- [x] `useContext`: Context API

### Additional Hooks
- [x] `useReducer`: Complex state logic
- [x] `useCallback`: Memoizing functions
- [x] `useMemo`: Memoizing values
- [x] `useRef`: DOM access & mutable values
- [x] `useImperativeHandle`: Customizing exposed refs
- [x] `useLayoutEffect`: Synchronous DOM updates
- [x] `useDebugValue`: Custom hook debugging

### React 18+ Hooks
- [x] `useId`: Unique IDs for accessibility
- [x] `useTransition`: Non-blocking state updates
- [x] `useDeferredValue`: Deferring UI updates
- [x] `useSyncExternalStore`: Subscribing to external stores
- [x] `useInsertionEffect`: CSS-in-JS injection

## 3. Advanced Patterns
- [x] **Custom Hooks**: Extracting reusable logic
- [x] **Context Pattern**: Global state management
- [x] **Compound Components**: Flexible component APIs (e.g., `<Select><Option /></Select>`)
- [x] **Render Props**: Sharing code between components
- [x] **Higher-Order Components (HOCs)**: Component logic reuse (Legacy but useful)
- [x] **Portals**: Rendering outside the DOM hierarchy
- [x] **Error Boundaries**: Catching runtime errors
- [x] **Forward Refs**: Passing refs to child components

## 4. Performance Optimization
- [x] **React.memo**: Preventing unnecessary re-renders
- [x] **Code Splitting**: `React.lazy` and `Suspense`
- [x] **Virtualization**: Rendering large lists efficiently
- [x] **Bundle Analysis**: Reducing app size
