JavaScript: map, filter, reduce (Beginner Friendly Docs)
This document explains map, filter, and reduce using simple code, dry runs, and tables.
________________


1. map()
What map does
                                                                     * Changes each item of an array
                                                                     * Returns a new array
                                                                     * Array length stays the same
________________


Syntax
array.map((currentItem) => {
  return newItem;
});


________________


Example
const numbers = [2, 4, 6];


const doubledNumbers = numbers.map((currentNumber) => {
  return currentNumber * 2;
});


console.log(doubledNumbers);


________________


Dry Run (map)
Round 1
                                                                     * currentNumber = 2
                                                                     * new value = 4
Round 2
                                                                     * currentNumber = 4
                                                                     * new value = 8
Round 3
                                                                     * currentNumber = 6
                                                                     * new value = 12
________________

Step
	currentNumber
	New Value
	Start

	1
	2
	4
	2
	4
	8
	3
	6
	12
	________________


Final Output
[4, 8, 12]

2. filter()
What filter does
                                                                     * Keeps items that match a condition
                                                                     * Removes unwanted items
                                                                     * Returns a new array
________________


Syntax
array.filter((currentItem) => {
  return trueOrFalse;
});


________________


Example
const numbers = [2, 4, 6];


const filteredNumbers = numbers.filter((currentNumber) => {
  return currentNumber > 3;
});


console.log(filteredNumbers);


________________


Dry Run (filter)
Round 1
                                                                     * currentNumber = 2
                                                                     * condition: 2 > 3 → false → remove
Round 2
                                                                     * currentNumber = 4
                                                                     * condition: 4 > 3 → true → keep
Round 3
                                                                     * currentNumber = 6
                                                                     * condition: 6 > 3 → true → keep

3. reduce()
What reduce does
                                                                     * Combines all items into one value
                                                                     * Uses an accumulator
________________


Syntax
array.reduce((accumulator, currentItem) => {
  return newAccumulator;
}, initialValue);


________________


Example
const numbers = [2, 4, 6];


const totalSum = numbers.reduce(
  (currentTotal, currentNumber) => {
    return currentTotal + currentNumber;
  },
  0
);


console.log(totalSum);

                                                                     * new total = 6
Round 3
                                                                     * currentTotal = 6
                                                                     * currentNumber = 6

4. async / await
What async / await does
                                                                     * Used to handle asynchronous work (time-taking tasks)
                                                                     * Makes async code look like normal step-by-step code
                                                                     * Commonly used with API calls, timers, database calls
________________


Simple Meaning (Easy words)
                                                                     * async → this function will take some time
                                                                     * await → wait here until the work is finished
________________


Basic Syntax
async function myFunction() {
  const result = await someAsyncTask();
  return result;
}


________________


Example (Very Simple)
function getNumber() {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(10);
    }, 1000);
  });
}


async function showResult() {
  const value = await getNumber();
  console.log(value);
}


showResult();


________________


Dry Run (async / await)
Step 1
                                                                     * showResult() is called
Step 2
                                                                     * await getNumber()
                                                                     * JS waits here
Step 3 (after 1 second)
                                                                     * Promise resolves with value = 10
Step 4
                                                                     * value = 10
                                                                     * console.log(10)
________________


Step Table
Step
	Action
	Value
	Start
	function called
	—
	1
	await getNumber()
	waiting
	2
	Promise resolved
	10
	3
	console.log
	10
	________________


Final Output
10


________________


async / await with API-like Example
async function getUser() {
  const response = await fetch('https://api.example.com/user');
  const data = await response.json();
  return data;
}


________________


Important Rules
                                                                     * await works **only inside async function`
                                                                     * async function always returns a Promise
________________


Compare: Promise vs async/await
Promise
	async/await
	.then()
	await
	.catch()
	try / catch
	Hard to read
	Easy to read
	________________


Easy Memory Rule
                                                                     * async → "this function waits"
                                                                     * await → "wait here"
