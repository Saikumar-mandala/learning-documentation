# HTML Interview Questions (A to Z)

## A - Alt Attribute
**Q: Why is the `alt` attribute important for images?**
A:
1. **Accessibility:** Screen readers read it to blind users.
2. **SEO:** Search engines use it to understand the image.
3. **Fallback:** It shows text if the image fails to load.

## D - DOCTYPE
**Q: What is `<!DOCTYPE html>`?**
A: It is not an HTML tag. It is an instruction to the browser about what version of HTML the page is written in. `<!DOCTYPE html>` tells the browser to use HTML5.

## H - HTML5
**Q: What are some new features in HTML5?**
A:
- **Semantic Tags:** `<header>`, `<footer>`, `<nav>`, `<section>`, `<article>`.
- **Form Controls:** Date pickers, email validation.
- **Multimedia:** `<audio>`, `<video>`.
- **Local Storage:** Saving data in the browser.

## M - Meta Tags
**Q: What are Meta Tags?**
A: They provide metadata about the HTML document. They are not displayed on the page but are machine-parsable.
Examples:
- `<meta charset="UTF-8">` (Character encoding)
- `<meta name="viewport" ...>` (Responsive design)
- `<meta name="description" ...>` (SEO description)

## S - Semantic HTML
**Q: What is Semantic HTML?**
A: It means using HTML tags that have meaning.
- **Bad:** Using `<div>` for everything.
- **Good:** Using `<header>` for the top, `<nav>` for links, `<button>` for buttons.
It helps with Accessibility and SEO.

## T - Title Attribute
**Q: What is the `title` attribute?**
A: It provides extra information about an element. It is usually shown as a tooltip text when the mouse moves over the element.

## W - Web Storage
**Q: What is the difference between LocalStorage and SessionStorage?**
A:
- **LocalStorage:** Data stays forever until deleted. (Even if you close the browser).
- **SessionStorage:** Data is lost when you close the browser tab.
